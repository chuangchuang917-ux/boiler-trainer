📖 使用方法
① 閱讀模式 — 點左側選單進入各燃料，展開各站步驟。藍色代號可 hover 查看設備名稱。
② 考試模式 — 開啟「考試模式 ON」，所有數值變為 ???，點擊揭示答案。
③ 比較模式 — 點「三燃料比較」，快速找出三種燃料的相同/差異步驟。
④ 查詢代號 — 點「設備代號查詢」，輸入任意代號（如 HV1402）即可查到中文名稱與設定值。
⑤ 全載數據 — 點「全載數據速查」，可即時查閱模擬機達到 Stable Run 狀態下的設定值，亦支援考試模式。
⑥ 學科模擬考 — 點「模擬考（80題）」，完全模擬正式學科上機考試，包含計時器與自動計分。
⑦ 分類練習 — 點「分類練習」，可按科目或工作項目針對性刷題，答錯自動記入錯題本。
⑧ 錯題本 — 點「錯題本」，集中複習所有曾答錯的題目，直到全部答對為止。
